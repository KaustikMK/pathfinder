# Pathfinder

Auto-generate macros for levels using simulation!

# ONLY SUPPORTS 1.7 LEVELS
### None of these are supported
- Duals
- Upside-down Slopes
- Partially Rotated Objects (not 90 degrees)
- 4x Speed
- Wave Mode
- Robot Mode
- Spider Mode
- Swing Mode
- Any Non-Visual Triggers
- Red Pads / Red Orbs
- Dash Orbs
- Teleport Portals
- Anything from 2.2

# Report Bugs
